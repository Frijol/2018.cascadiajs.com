## Website

CascadiaJS is powered by [GatsbyJS](https://www.gatsbyjs.org/).

### Install

Make sure that you have the Gatsby CLI program installed:
```sh
npm install --global gatsby-cli
```

Then you can run the dev server by:
```sh
gatsby develop
```
